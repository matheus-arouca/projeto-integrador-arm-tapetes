# Projeto - ARM Tapetes

Projeto acadêmico em desenvolvimento na disciplina Projeto Integrador I do curso de Análise e Desenvolvimento de Sistemas da FATEC Mogi das Cruzes.

## Objetivo

Desenvolver um website para a empresa ARM Tapetes, com foco em melhorar a visibilidade online, facilitar a personalização de tapetes, organizar pedidos e automatizar parte do processo de vendas.

## Status do projeto

Em desenvolvimento.

## Funcionalidades previstas

- Apresentação da empresa e dos produtos
- Personalização de tapetes
- Solicitação de orçamento
- Organização de pedidos
- Informações sobre pagamento e entrega

## Conteúdos praticados

- Engenharia de Software
- Levantamento de requisitos
- Documentação de software
- Análise de negócio
- Trabalho em equipe
- Desenvolvimento web
